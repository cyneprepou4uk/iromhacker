Вкладка View

Можно переключиться на 4 разных режима отображения памяти.

В соответствующих статьях будет подробно рассказано об этих вкладках и их возможностях.

NES Memory

Отображение памяти центрального процессора. Включает в себя RAM, регистры, батарейку (при ее наличии) и память ROM'а, подгруженную из ROM File. Размер окна $0000-$FFFF.

PPU Memory

Отображение памяти видеопроцессора. Включает в себя графику фона и спрайтов, атрибуты фона и палитру. Размер окна $0000-$3FFF.

OAM Memory

Отображение памяти спрайтов. Размер окна минимум $0000-$00FF.

ROM File

Отображение содержимого файла ROM'а. Аналогично открытию файла в другом Hex-редакторе. Размер окна зависит от размера файла.
