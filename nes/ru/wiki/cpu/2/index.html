Список инструкций процессора

Список неофициальных инструкций можно посмотреть на NesDev. Некоторые из них можно использовать для небольшой оптимизации своего кода.

Выбери интересующую инструкцию из списка, чтобы посмотреть ее описание и примеры использования.

ADC AND ASL BCC BCS BEQ BIT BMI BNE BPL BRK BVC BVS CLC
CLD CLI CLV CMP CPX CPY DEC DEX DEY EOR INC INX INY JMP
JSR LDA LDX LDY LSR NOP ORA PHA PHP PLA PLP ROL ROR RTI
RTS SBC SEC SED SEI STA STX STY TAX TAY TSX TXA TXS TYA

LDA (Load Accumulator)

Загружает в регистр A фиксированный байт или байт из адреса. Байт в регистре будет перезаписан.

Флаги

После выполнения команды обновляется состояние флагов.

Z

Если загруженный байт = #$00, то Z = 1, иначе Z = 0.

N

Если загруженный байт в диапазоне #$00-#$7F, то N = 0, иначе N = 1.

STA (Store Accumulator)

Записывает байт из регистра A в указанный адрес.

Состояние флагов не обновляется. Байт в регистре A остается без изменений.

STX (Store X Register)

Записывает байт из регистра X в указанный адрес.

Состояние флагов не обновляется. Байт в регистре X остается без изменений.

STY (Store Y Register)

Записывает байт из регистра Y в указанный адрес.

Состояние флагов не обновляется. Байт в регистре Y остается без изменений.
